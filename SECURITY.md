# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public GitHub issue.**

Instead, send an email to: **jenewland1999@gmail.com**

Please include:

- A description of the vulnerability
- Steps to reproduce (if applicable)
- Any potential impact
- Suggested fix (if you have one)

## Response

You can expect an initial response within 72 hours. The issue will be investigated and, if confirmed, a fix will be released as soon as practical.

## Scope

This policy applies to the PIM Role Activator CLI codebase. Issues with Azure PIM itself or the Azure SDK should be reported to Microsoft.
